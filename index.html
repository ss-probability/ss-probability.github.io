Definitions

Sample Space

In probability theory, the sample space of an experiment or random trial is the set of all possible outcomes or results of that experiment. A sample space is usually denoted using set notation, and the possible ordered outcomes are listed as elements in the set.
A sample space is called a finite sample space if it has a finite number of outcomes in it. If there are N outcomes in S, we can identify them as S1, S2, ..., SN. This would led us to write S as
S = {S1, S2, ..., SN}

Probability

Probability is the bridge betweeen desription of samples and inferences about populations, because it is both the language and the measure of uncertainty.

Examples

Example 1

Consider tossing a die and then recording the number that comes up. The set of possible outcomes is defined in the set S:
S = {1, 2, 3, 4, 5, 6}

Let A be the event that an odd number occurs. B that an even number occurs and C that a perfect square occurs:
A = {1, 3, 5} B = {2, 4, 6} 
C = {1, 4}

Then, B ∪ C = {1, 2, 4, 6} is the event that an even number or a square number occurs;
A ∩ B = {1} is the event that a perfect square odd number occurs;
C' = {2, 3, 5, 6} is the event that a perfect square number does not occur,
Note that A and B are mutually exclusive, A ∩ B = Ø, because an even number and an odd number cannot occur simultaneously.

Example 2

Two coins are tossed simultaneously. Then the possible outcomes are
S = {HH, HT, TH, TT}

Let A be the event that a head appears on any coin and B that all the tosses are the same:
A = {HT, TH, HH} B = {HH, TT}

Then A ∩ B = {HH} is a simple event in which only heads appear. The event that 3 heads appear is the empty set Ø.

Example 3

If repetition of digit is not allowed, how many three-digit numbers can be formed from the digits 1, 2, 4, 5, and 6? How many of these are odd?
A three-digit number is composed of hundred's, ten's, and unit's digit. Hence by multiplication rule there are
5 ⋅ 4 ⋅ 3 = 60 three-digit numbers.
Let A be the event that a head appears on any coin and B that all the tosses are the same:
4 ⋅ 3 ⋅ 2 = 24 three-digit odd numbers.

Example 4

Picking 2 marbles, one at a time, from a bag that contains many blue and red marbles.
Possible outcomes are: (Blue, Blue), (Blue, Red), (Red, Blue) and (Red, Red).

S = {(B,B), (B,R), (R,B), (R,R)}

Probability is the extent to which an event is likely to occur, measured by the ratio of the favorable cases to the whole number of cases possible.
